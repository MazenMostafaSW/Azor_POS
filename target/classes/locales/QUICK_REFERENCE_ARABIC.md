## Usage Tips

1. **Consistency**: Always use the same Arabic term for the same English word
2. **Context**: Some words have different translations based on context (e.g., Receipt vs Ticket)
3. **Formality**: All translations use formal Modern Standard Arabic
4. **Numbers**: Arabic numerals (0-9) are used, not Arabic-Indic numerals (٠-٩)

## Common Phrases

### Confirmation Dialogs
- "هل أنت متأكد من رغبتك في..." = "Are you sure you want to..."
- "سيتم فقدان التغييرات" = "Changes will be lost"
- "هل تريد المتابعة؟" = "Do you wish to continue?"

### Success Messages
- "تم بنجاح" = "Successfully completed"
- "تم الحفظ" = "Saved"
- "تم الإغلاق" = "Closed"

### Error Messages
- "غير قادر على..." = "Unable to..."
- "خطأ في..." = "Error in..."
- "لم يتم العثور على..." = "Not found..."
